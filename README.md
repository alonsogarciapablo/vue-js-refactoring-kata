# Vue.js Refactoring Kata


## Setting up the project


1. Checkout the project

```bash
git clone https://github.com/alonsogarciapablo/vue-js-refactoring-kata.git

```

2. Install dependencies

```bash
npm install

```

3. Run the tests

```bash
npm run test

```

4. Run the app

```bash
npm run serve
```

5. You're good to go! 👏
