# Vue.js Refactoring Kata


## Setting up the project


1. Checkout the project

```
git clone https://github.com/alonsogarciapablo/vue-js-refactoring-kata.git

```

2. Install dependencies

```
npm install

```

3. Run the tests
```
npm run test

```

4. Run the app
```
npm run serve
```

5. You're good to go! 👏
